# Local Frontend Server for YOLO Project

This server allows you to serve the frontend directory with CORS enabled for local development and testing.

## Usage

1. Make sure you have Python 3 installed.
2. Run this script from the project root or the frontend directory:

    python serve_frontend.py

3. Open your browser to http://localhost:8080

---

This script uses Python's built-in HTTP server and adds CORS headers for local testing.
